# 使用npm构建的小程序模板

### 此模板已使用 vant 组件
* 默认已关闭引入索引打印

#### 第一步:
运行 `npm i` 下载依赖包

#### 第二步:
> 击开发者工具中的菜单栏：工具 --> 构建 npm

#### 第三步:
> 勾选 “使用 npm 模块” 选项：

 ### typings
* 用来 webstrom 开发提示使用 不需要可以自行删除
* 设置方法
 settings --> Languages&Frameworks --> JavaScript --> Libraries

 ### utils
* ajaxPromise 基于Promise简单的请求封装
* formatTime 默认日期格式修改
